# tanakh-words

Check if your Hebrew text contains words from the Tanakh.

Type any Hebrew text and instantly see which words appear in the Torah, Nevi'im, or Ketuvim, with verse references, word frequency, and section filtering.

**[Live Site](https://idantravitsky.github.io/tanakh-words/)**

## מילים בתנ״ך

בדקו אם המילים שלכם מופיעות בתנ״ך.

הקלידו טקסט בעברית וראו מיד אילו מילים נמצאות בתורה, בנביאים או בכתובים, עם מקורות פסוקים, תדירות וסינון לפי חלק.

Tanakh text sourced from [Sefaria](https://www.sefaria.org).
