# tanakh-words

"None of these words are in the Bible" but for the Tanakh.

Paste any Hebrew text and instantly see which words actually appear in the Tanakh. Inspired by the [internet meme](https://en.wiktionary.org/wiki/none_of_these_words_are_in_the_Bible).

**[Try it](https://idantravitsky.github.io/tanakh-words/)**

## מילים בתנ״ך

הגרסה העברית/תנ״כית של המים "none of these words are in the Bible".

הדביקו טקסט בעברית ותראו כמה מהמילים באמת מופיעות בתנ״ך.

Tanakh text sourced from [Sefaria](https://www.sefaria.org).
